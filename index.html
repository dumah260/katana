Sorting

  • Insertion Sort

    Insertion sort is a method of sorting an array by dividing the array into a 'sorted' part and 'unsorted' part.

    Each element from the unsorted part are compared with the elements in the sorted part ,and then inserted in the correct position in the sorted part.

                                            
function insertionSort(array){

    let i;
    let h;

    // select the 'unsorted' elements
    for (i = 1; i < array.length; i++){ //store the current 'unsorted' element value

        let tmp = array[i]; //loop through the items in the sorted array
        for (h = i-1; h >= 0; h--){

            //if the 'unsorted' element is smaller than 'sorted' element, swift other elements to the right to make the proper position for 'unsorted' element
            if (array[h] > tmp){
                array[h+1] = array[h];
            } else {
                break;
            }
        }

        // insert the 'unsorted' element to its proper position
        array[h+1] = tmp;

    }

    return array;
}

  • Selection Sort

    Selection Sort consists in a basic idea. For each position in an array, check the remaining positions to find the smaller value. If there is a value smaller than the current position value, swap them.

                                            
function selectionSort (array) {

    const arrayLenght = array.length; //number of items in the array

    let firstCounter = 0; //counter for the first loop
    let secondCounter = 0; //counter for the second loop
    let minIndex = 0; //index of the small value of the array
    let aux = 0; //variable to keep the value that have to change position

    for (firstCounter; firstCounter < arrayLenght; firstCounter++) {

        minIndex = firstCounter; //start the min index with the current loop index
        secondCounter = firstCounter + 1; //the second loop should start from the next postion

        for (secondCounter; secondCounter < arrayLenght; secondCounter++) { //check if the value in the second loop is smaller than the current min value
            if (array[minIndex]> array[secondCounter]) {
                minIndex = secondCounter; //if is smaller change the min index for the second loop index
            }
        }

        //if the min index changed we need to swap
        if (firstCounter != minIndex) {
            aux = array[ firstCounter ]; //tem variable receives the current value of the fisrt loop
            array[ firstCounter ] = array[ minIndex ]; //the array receives the smaller value in the first loop position
            array[ minIndex ] = aux; //the index that had the smaller value receives the value of the first loop position
        }
    }
    return array;
}

  • Merge Sort

    Divide and conquer. The first step is divide all the array postions in the middle recursively. Once divided, all the positions are merged swapping values when necessary. 

                                            
function mergeSort (array) {

    let middlePosition = 0;
    let leftItems = [];
    let rightItems = [];

    //if the array have more than 1 postion we need to split it
    if (array.length > 1) {

        middlePosition = array.length / 2; // divide the number of postions in 2
        middlePosition = Math.floor(middlePosition); // get the integer result

        leftItems = array.slice(0, middlePosition); //receives the first part of the array
        rightItems = array.slice(middlePosition); //receives the second part of the array

        //call the function again to keep dividing the array
        leftItems = mergeSort(leftItems);
        rightItems = mergeSort(rightItems);

        return mergeItems(leftItems, rightItems); //put both sides of the array together swapping when necessary
    } else {
        return array;
    }
}

function mergeItems(leftItems, rightItems) {

    let arraySorted = []; //new array with merged values

    while (leftItems.length && rightItems.length) {

        //check if the item in the left side is smaller than the item in the right side
        if (leftItems[0] < rightItems[0]) {
            arraySorted.push(leftItems[0]); //move the item to the new sorted array
            leftItems.shift(); // remove the item in the first position
        } else {
            arraySorted.push(rightItems[0]); //move the item to the new sorted array
            rightItems.shift(); // remove the item in the first position
        }
    }

    //concat the newsorted array with the remaining items
    arraySorted = arraySorted.concat(leftItems).concat(rightItems);

    return arraySorted;
}
